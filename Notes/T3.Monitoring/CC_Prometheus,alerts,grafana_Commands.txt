## prometheus + blackbox

#1. Open terminal 1 on blackbox in: 
C:\Program Files\blackbox_exporter-0.25.0.windows-amd64
#2. in terminal blackbox type:
docker run -d -p 9115:9115 quay.io/prometheus/blackbox-exporter
#3.Open terminal 2 on prometheus on: 
C:\Program Files\prometheus-2.51.2.windows-amd64
#4. in terminal prometheus type:
code .
#5. Drag & Drop the prometheus-exam.yml file in C:\Program Files\prometheus-2.51.2.windows-amd64 and rename if needed after
#6. # Add URL in .yml
#7. in terminal prometheus type:   
prometheus.exe --config.file prometheus-exam.yml
#8. Check if prometheus works on: localhost:9090

##AlertManager
1.Open terminal on AlertManager in:
C:\Program Files\alertmanager-0.27.0.windows-amd64
2. in terminal AlertManager type:
code .
3.Drag & Drop the alertmanager-exam.yml file in C:\Program Files\alertmanager-0.27.0.windows-amd64 and rename if needed after
4.go to https://webhook.site and get your unique url, it will be used in the alertmanager-exam.yml
5. in terminal AlertManager type:
alertmanager.exe --config.file alertmanager-exam.yml
6. Check if AlertManager works on: localhost:9093

//add alert to prometheus:
7.Drag & Drop the "alert-rules-exam.yml" file in C:\Program Files\prometheus-2.51.2.windows-amd64 and rename if needed after
8.Turn on: the #alertmanager comments in prometheus-exam.yml
9. in terminal prometheus type:   
prometheus.exe --config.file prometheus-exam.yml
10.Check if we have Alerts in prometheus on localhost:9090
11.Check if we have Alerts in AlertManager on localhost:9093
